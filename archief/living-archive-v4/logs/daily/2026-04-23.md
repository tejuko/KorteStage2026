## Wat heb ik geleerd?
- Interne linking is een SEO-bouwsteen: relevante anchor-text + logische verbinding tussen verwante pagina's helpt zoekmachines (en gebruikers) navigeren
- Een product-tag pagina heeft geen running text → daar past geen contextuele link; per pagina-type bepalen of een link überhaupt zinvol is
- Kwaliteitsbewaking: niet klakkeloos een link plaatsen omdat het in de spreadsheet staat — context van de pagina is leidend
