### Fix 1 — FOUC op Home Dark (live)

**Root cause:** de plugin **"Speed Up – Optimize CSS Delivery"** converteert alle stylesheets naar async `rel="preload"`, waardoor de pagina kort zonder CSS rendert (flash of unstyled content).

**Wijziging in `inc/dark-function.php`** — drie filters toegevoegd om de dark-styles uit te sluiten van de async-conversie:

```php
add_filter('rocket_rucss_external_exclusions', function($exclusions) {
    $exclusions[] = 'markup/dist/css';
    return $exclusions;
});

add_filter('rocket_rucss_inline_atts_exclusions', function($exclusions) {
    $exclusions[] = 'dark-critical-fouc';
    return $exclusions;
});

add_filter('speed-up-optimize-css-delivery', function($handle) {
    if (in_array($handle, ['dark-styles', 'dark-styles-fonts'], true)) {
        return true;
    }
    return $handle;
});
```

**Wijziging in `header-dark.php`** — de `<style>`-tag met de body-background verplaatst naar ná `wp_head()` met een id en `!important`:

```php
<?php wp_head(); ?>
<style id="dark-critical-fouc">body{background:#000618!important;color:#fff!important}</style>
```

**Deploy-procedure:**
1. WP Rocket → *Clear Used CSS* voor de Dark-pagina
2. WP Rocket → *Clear Cache*
3. Verificatie in incognito → witte flits weg

### Fix 2 — SEA-label verwijderen bij DAS Import (post 32966)

**Type fix:** database-only (geen bestandsaanpassingen).

**SQL-queries** (via WP-CLI of phpMyAdmin/Adminer) op de live DB:

```sql
-- 1. Verlaag label-count van 3 naar 2
UPDATE wp_postmeta SET meta_value='2'
WHERE post_id=32966 AND meta_key='modules_2_labels';

-- 2. Verwijder SEA (index 1)
DELETE FROM wp_postmeta
WHERE post_id=32966 AND meta_key IN ('modules_2_labels_1_label','_modules_2_labels_1_label');

-- 3. Schuif UX (index 2) op naar index 1
UPDATE wp_postmeta SET meta_key='modules_2_labels_1_label'
WHERE post_id=32966 AND meta_key='modules_2_labels_2_label';

UPDATE wp_postmeta SET meta_key='_modules_2_labels_1_label'
WHERE post_id=32966 AND meta_key='_modules_2_labels_2_label';
```

Daarna: WP Rocket cache leegmaken voor Home Dark.
**Verificatie:** DAS Import sectie toont nu SEO + UX (geen SEA meer).

### Fix 3 — Team-card overlap + gelijke afstanden

**Twee CSS-aanpassingen** in `markup/dist/css/style.css`:

```css
/* Fix 1 — Rico Toet (card 2) overlapping Yannick */
@media (min-width: 1388px) {
  .team-card-list > li:nth-child(2) {
    transform: translate(0, 40px); /* was: translate(-50px, 40px) */
  }
}

/* Fix 2 — Jordy de Boer (card 4) te ver naar rechts */
@media (min-width: 1388px) {
  .team-card-list > li:nth-child(4) {
    transform: translate(0, 164px); /* was: translate(30px, 164px) */
  }
}
```

**Wat dit doet:** de horizontale offsets (-50px / +30px) op brede schermen geneutraliseerd. Het verticale stagger-effect blijft intact; kaarten houden hun flex-gap van 16px als horizontale afstand. Geen overlap, gelijke tussenruimtes.

Daarna: WP Rocket cache leeg.
