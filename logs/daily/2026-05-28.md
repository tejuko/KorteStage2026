| Element | Gelocked op |

## Wat heb ik geleerd?
- **Een header die "schiet" bij scroll is bijna nooit één property.** De parent-theme `.nav-colored`-class wijzigde padding, gap, logo-width én header-padding tegelijk. De fix was niet één waarde aanpassen, maar **alle states gelijktrekken** zodat er niets meer te animeren valt.
- **`transition: none !important` als laatste redmiddel** tegen "sneaky" animaties die je niet makkelijk kunt traceren naar één regel — niet elegant, maar effectief en goed afgebakend.
- **Scopen via een bestaande `is_singular()`-conditional** is een veilige manier om op een productiesite (LIVE!) een fix te doen zonder andere pagina's te raken — mits je backup maakt en cache leegt.
- **Volle-breedte gradientbanden zonder horizontale scrollbar:** `left:-100vw; right:-100vw` op een pseudo-element + `overflow-x:clip` op de container geeft een band die "buiten" de content doorloopt zonder layout te breken.
- **Een herontwerp bouw je in lagen op:** baseline (wit) → per-blok accent (lokale band) → component-styling (alternerende DARK/LIGHT cards) → micro-interactie (hover-slide, fade-up). In stappen werken houdt het overzichtelijk.
